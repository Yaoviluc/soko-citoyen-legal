6. SOKO Bouclier (SOS personnel)

6.1 Activation

Trois mécanismes : bouton principal, secouage 3× rapide, code vocal personnalisé.

6.2 Fonctionnement

  • Partage de position GPS + audio chiffré avec vos contacts de confiance désignés
  • Escalade automatique aux autorités locales (police, gendarmerie) selon votre pays
  • Notification persistante Android (foreground service) tant que l'alerte est active

6.3 Faux SOS

Le déclenchement intentionnel d'un faux SOS est passible :

  • 🇧🇯 Bénin : article 528 Code pénal (entrave aux secours)
  • 🇸🇳 Sénégal : article 247 Code pénal (fausse alerte aux secours)
  • 🇨🇮 Côte d'Ivoire : article 304 Code pénal
  • Sanctions similaires dans les autres pays

SOKO collaborera avec les autorités sur réquisition légale.

7. SOKO Retrouve (alerte enfant disparu)

7.1 Activation

  • Nécessite une vérification d'identité niveau 2 (selfie + ID) du parent
  • L'alerte concentrique se déclenche à 5km, puis 15km, puis 30km autour du dernier lieu vu

7.2 Suppression automatique

Photos d'enfants automatiquement supprimées 30 jours après résolution de l'alerte.

7.3 Respect des principes UNICEF

Conformité aux principes UNICEF sur la protection numérique de l'enfance : pas de partage commercial, pas de profilage marketing, suppression définitive après usage.

8. Constat Amiable Numérique

8.1 Cadre

SOKO permet aux conducteurs impliqués dans un accident léger sans blessure grave de remplir un constat amiable numérique conforme aux exigences de la CIMA (Conférence Interafricaine des Marchés d'Assurances), opérant dans 14 pays d'Afrique.

8.2 Valeur juridique

  • Signature électronique avec preuve géographique GPS (les deux parties doivent se trouver à proximité du lieu d'accident)
  • Hash SHA-256 du document pour intégrité
  • Acceptation par les assureurs CIMA partenaires : NSIA, SAHAM, SUNU, Allianz Africa, AXA, Atlantique, Sonar, AGF, GTA-C2A, Wafa

8.3 Limites

  • En cas de blessure grave, blessure mortelle, ou désaccord persistant : la procédure judiciaire normale s'applique
  • L'app NE remplace PAS le rapport de police obligatoire si requis par la loi locale
