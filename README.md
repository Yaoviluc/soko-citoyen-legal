# SOKO Citoyen — Documents légaux

Site statique hébergeant la Politique de confidentialité et les Conditions d'utilisation de **SOKO Citoyen**, la super-application 21 pays CEDEAO + CEMAC.

🌍 **Pays couverts** : Bénin, Burkina Faso, Cabo Verde, Côte d'Ivoire, Gambie, Ghana, Guinée, Guinée-Bissau, Libéria, Mali, Mauritanie, Niger, Nigeria, Sénégal, Sierra Leone, Togo, Cameroun, Centrafrique, Congo, Gabon, Guinée Équatoriale, Tchad.

## URLs publiques

- 🌐 **Accueil** : https://yaoviluc.github.io/soko-citoyen-legal/
- 🔒 **Politique de confidentialité** : https://yaoviluc.github.io/soko-citoyen-legal/privacy.html
- 📜 **Conditions d'utilisation** : https://yaoviluc.github.io/soko-citoyen-legal/terms.html

## Source des documents

Les fichiers Markdown source sont maintenus dans le repo principal `soko-citoyen` à `docs/legal/`. Le script `build.mjs` ici les convertit en HTML branded SOKO.

## Régénérer les pages

```sh
npm install
node build.mjs
```
