Politique de confidentialité

Comment SOKO traite vos données · Version 1.0 · 14 mai 2026

Préambule

SOKO Citoyen ("SOKO", "nous", "notre application") est une plateforme multi-services destinée aux populations d'Afrique de l'Ouest et d'Afrique Centrale, présente dans 21 pays :

  • CEDEAO (15) : Bénin, Burkina Faso, Cabo Verde, Côte d'Ivoire, Gambie, Ghana, Guinée, Guinée-Bissau, Libéria, Mali, Niger, Nigeria, Sénégal, Sierra Leone, Togo
  • CEMAC (6) : Cameroun, Centrafrique, Congo, Gabon, Guinée Équatoriale, Tchad

Cette politique respecte les législations applicables dans ces 21 pays, dont notamment :

  • 🇧🇯 Bénin — Loi n° 2017-20 portant Code du Numérique (APDP)
  • 🇸🇳 Sénégal — Loi n° 2008-12 sur la protection des données personnelles (CDP)
  • 🇨🇮 Côte d'Ivoire — Loi n° 2013-450 (ARTCI)
  • 🇧🇫 Burkina Faso — Loi n° 010-2004 (CIL)
  • 🇬🇭 Ghana — Data Protection Act 2012 (Act 843) — DPC
  • 🇳🇬 Nigeria — Nigeria Data Protection Act 2023 — NDPC
  • 🇨🇲 Cameroun — Loi n° 2010/012 (ANTIC)
  • 🇲🇱 Mali — Loi n° 2013-015 (APDP)
  • 🇹🇬 Togo — Loi n° 2019-014 (IPDCP)
  • 🇬🇦 Gabon — Loi n° 001/2011 (CNPDCP)
  • 🇳🇪 Niger — Loi n° 2017-28 (HAPDP)
  • Conformité substantielle au Règlement Général sur la Protection des Données (RGPD) quand un utilisateur en Diaspora est concerné

2. Comment vos données sont utilisées

2.1 Pour fournir les services demandés

  • SOKO Sécurité (Bouclier SOS, Retrouve enfants disparus, Témoin anonyme, Route signalements, Constat Amiable) : géolocalisation + contacts choisis
  • SOKO Marketplace : photos et description de vos annonces
  • SOKO Transport / Bus : recherche d'itinéraires + réservations
  • SOKO Move : matching covoiturage selon votre direction (passenger ou driver)
  • SOKO Pay : transactions Mobile Money via opérateurs locaux
  • SOKO Santé : carnet de vaccination, alertes proches, consultations
  • SOKO École : suivi élèves pour parents + cockpit directeur
  • SOKO Agriculture, Foncier, Tontine, Diaspora, … : selon usage actif

2.2 Pour personnaliser votre expérience

Le Moteur d'Intelligence SOKO (MIS) réorganise votre écran d'accueil selon les modules que vous utilisez le plus. Cette personnalisation reste sur votre appareil — nous ne construisons pas de profil publicitaire.

2.3 Pour prévenir la fraude

  • Détection automatique : > 200 ventes/heure, GPS variant de > 500m vs adresse boutique, changements de prix > 10/jour
  • Validation OTP + vérification KYC pour paiements > 200 000 FCFA (loi CIMA / UEMOA)
  • Hash des PIN PBKDF2 100 000 itérations (jamais en clair)

2.4 Pour respecter nos obligations légales

  • Conservation comptable des paiements : 10 ans (loi UEMOA / CEMAC)
  • KYC SOKO Pay : conservation des copies de pièces d'identité par notre prestataire KYC (Smile ID ou équivalent) selon les seuils CIMA :
    • L0 (téléphone vérifié) — jusqu'à 200 000 FCFA / mois
    • L1 (CNI / passeport) — jusqu'à 2 000 000 FCFA / mois
    • L2 (selfie + ID match) — jusqu'à 10 000 000 FCFA / mois
    • L3 (entreprise + KBIS équivalent) — illimité
  • Conservation logs d'audit : 1 an

3.4 Autorités publiques

Uniquement sur réquisition légale écrite (police, gendarmerie, justice) de votre pays de résidence. Nous documentons toute requête dans un registre interne d'accès aux données (article 21 Loi Bénin 2017-20, articles équivalents dans les autres pays).

4. Sécurité

  • 🔐 TLS 1.3 pour toutes les communications client ↔ serveur
  • 🔐 AES-256-GCM pour les documents stockés dans le Coffre-fort (clé dérivée de votre PIN, jamais transmise)
  • 🔐 PBKDF2-HMAC-SHA-256 100 000 itérations pour le hash de votre PIN serveur
  • 🔐 expo-secure-store (Android Keystore / iOS Keychain) pour le token de session
  • 🔐 Row Level Security sur 155+ tables Supabase — vous ne pouvez voir que vos propres données
  • 🔐 Device binding : vous pouvez voir et déconnecter à distance chaque appareil connecté à votre compte
  • 🔐 Connexion biométrique optionnelle (Face ID / empreinte) — clé biométrique reste sur votre appareil
  • 🔐 Foreground services Android avec notification visible permanente quand actifs (Bouclier en cours)

En cas de fuite de données nous concernant, nous notifierons les utilisateurs affectés et l'autorité de votre pays dans les 72 heures.

5. Vos droits

À tout moment, depuis l'application (Profil → Confidentialité) :

5.1 Droit d'accès et de portabilité

Exporter toutes vos données aux formats JSON, CSV ou PDF — incluant : profil, paiements, signalements, données médicales, scolarité, etc.

5.2 Droit de rectification

Modifier votre nom, téléphone, photo, langue, consentements depuis Profil.

5.3 Droit à l'effacement

Supprimer définitivement votre compte : Profil → Confidentialité → Supprimer mon compte → confirmer en tapant "SUPPRIMER MON COMPTE".

Une fois confirmé : suppression irréversible sous 30 jours (délai légal pour permettre annulation accidentelle). Conservation uniquement des données légalement obligatoires (paiements > 10 ans, signalements judiciaires).

5.4 Droit d'opposition

Désactiver tout ou partie de la collecte : Profil → Confidentialité → 3 toggles :

  • Données santé (OFF par défaut)
  • Localisation arrière-plan (OFF par défaut, sauf activation explicite Bouclier)
  • Statistiques d'usage anonymisées (ON par défaut, désactivable)

6. Mineurs

6.1 Âge minimum

Inscription possible à partir de 16 ans (ou 13 ans avec consentement parental écrit communiqué à luc.houndenou40@gmail.com).

6.2 Module SOKO Retrouve (alerte enfant disparu)

Les parents peuvent enregistrer leurs enfants mineurs pour bénéficier d'une alerte concentrique en cas de disparition (cercles 5km → 15km → 30km).

  • Aucune photo d'enfant n'est rendue publique tant qu'une alerte n'est pas activement déclenchée par le parent
  • Les alertes nécessitent un niveau 2 de vérification d'identité du parent (selfie + CNI/passeport)
  • Les photos sont automatiquement supprimées 30 jours après résolution de l'alerte
  • Conformité aux principes UNICEF de protection de l'enfance numérique

8. Cookies et traceurs

L'application mobile ne contient aucun cookie tiers ni traceur publicitaire. Seuls les identifiants techniques internes (anonymes, hachés) sont utilisés pour la sécurité et la prévention de la fraude.

Notre site web (https://soko-citoyen.app) utilise uniquement des cookies techniques nécessaires (préférence de langue, session admin) — aucun traceur Google Analytics, Facebook Pixel ou équivalent.
